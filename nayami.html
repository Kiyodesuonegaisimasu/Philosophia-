🌙不安でいっぱいの時

心配や恐れが頭を離れず、落ち着かない。そんな夜に。

将来が不安で、夜になると悪いことばかり考えてしまう時。
エピクロス →
思い通りにならないことに、イライラして疲れきった時。
ゼノン(ストア派) →
人の一言や出来事に、必要以上に振り回されて苦しい時。
エピクテトス →
根拠のあいまいな不安や噂に、飲み込まれそうな時。
ラッセル →
手に入らないものや失ったものに、心が囚われて苦しい時。
ブッダ(釈迦) →
「きっとこうなるはずだ」と決めつけて、思い込みで動いてしまった時。
デイヴィッド・ヒューム →
考えても答えの出ない問いで、頭がぐるぐるして苦しい時。

🪨頑張っても報われない・自分を責める時

努力しているのに空回り。つい自分を責めてしまう。

努力が足りないからだ、と自分を責めてすり減っている時。
マルクス →
自分にはたいした才能がない、と落ち込んだ時。
アリストテレス →
やめたいのにやめられない習慣に、自己嫌悪を感じた時。
アウグスティヌス →
自分の弱さや欲深さに嫌気がさした時。
荀子 →
自分の力ではどうにもならず、すべてを投げ出したくなった時。
親鸞 →
自分の性格や境遇を、どうしようもないと感じて苦しい時。
スピノザ →
欲しかったものを手に入れたのに、なぜか満たされなかった時。
ショーペンハウアー →

🤝人に振り回される・関係に悩む時

誰かの言動に心をかき乱される。人づきあいがしんどい。

人間関係がぎくしゃくして、どう振る舞えばいいか迷った時。
孔子 →
「あの人のことは分かってる」と決めつけて、すれ違った時。
レヴィナス →
身内びいきや えこひいきに、もやもやした時。
墨子 →
感情的な言い合いになって、話がまったく噛み合わなかった時。
ハーバーマス →
良かれと思って、つい他人の生き方に口を出したくなった時。
J.S.ミル →
ルールやマナーを守らない人を見て、うんざりした時。
ホッブズ →
人の善意をあてにして、期待が裏切られて疲れた時。
韓非子 →
人間なんて結局みんな自分勝手だ、と人間不信になった時。
孟子 →
誰かの苦しみから、つい目をそらしてしまった自分に気づいた時。
シモーヌ・ヴェイユ →

🪞自分を見失った・自分らしさに悩む時

周りに合わせるうちに、自分が分からなくなった。

言い合いになって、自分の意見をうまく説明できずモヤモヤした時。
ソクラテス →
世間の価値観に合わせているうちに、自分が分からなくなった時。
ルソー →
他人のことばかり気にして、自分を見失っていた時。
モンテーニュ →
「あなたはこういう人でしょ」と役割を押しつけられた時。
ボーヴォワール →
「自分はこういう人間だ」と型にはめて、窮屈になった時。
ドゥルーズ →
「らしくしなさい」という期待に、しんどさを感じた時。
ジュディス・バトラー →
周りに流されて、自分の頭で考えていないと気づいた時。
福沢諭吉 →
自分は生まれつきこういう人間だ、と決めつけて諦めかけた時。
ジョン・ロック →

🌌むなしい・何のために生きるか分からない時

すべてが無意味に思える。頑張る理由が見えない。

頑張っても意味がないと感じて、すべてが虚しくなった時。
カミュ →
なぜ自分がここにいるのか、何もかも無意味に思えた時。
ライプニッツ →
毎日をなんとなく過ごして、時間だけが流れていくと感じた時。
ハイデガー →
日々の忙しさに流されて、何のために頑張っているのか見失った時。
マルクス・アウレリウス →
「本当の幸せは、いつか遠い先にある」と感じて疲れた時。
空海 →
何かに没頭している時が、一番いきいきしていると気づいた時。
西田幾多郎 →
現実がうまくいかなくて、理想なんて意味がないと感じた時。
プラトン →
理屈では割り切れない感情を、うまく説明できず苦しい時。
パスカル →

🧭決められない・迷っている時

選択肢の前で立ちすくむ。何が正解か分からない。

誰も代わってくれない選択を前に、一人で怖くなった時。
キルケゴール →
「そうするしかなかった」と、自分に言い訳したくなった時。
サルトル →
二つの選択肢の間で、どちらも選べず身動きが取れない時。
ヘーゲル →
やることが多すぎて頭がぐちゃぐちゃ。何から手をつければいいのか分からない。そんな時。
デカルト →
みんなにとって何が正解か、決めきれず迷った時。
ベンサム →
自分だけ得しようとして、後ろめたさを感じた時。
カント →
信じたい気持ちと、理屈で考えた結論が食い違って迷った時。
トマス・アクィナス →
きれいごとだけでは物事が進まず、板挟みになった時。
マキャヴェッリ →
自分の立場からしか物事を見られていない、と気づいた時。
ロールズ →
自分の利益のために働くことに、どこか後ろめたさを感じた時。
アダム・スミス →

🍵頑張りすぎ・力を抜きたい時

全部抱え込んで、心も体もすり減っている。

頑張っているのに空回りして、力が入りすぎていると感じた時。
老子 →
悩みが頭を離れず、視野が狭くなっていると感じた時。
荘子 →
効率や時間に追われて、心がすり減っていると感じた時。
ベルクソン →

ハイエク →
何をやっても、見返りや結果ばかり気にしてしまう時。
道元 →
時間がないと言いながら、気づけば一日を無駄にしていた時。
セネカ →
毎日がバラバラで、生活にまとまりがないと感じた時。
ピュタゴラス →
問題に正面からぶつかって、消耗してしまった時。
孫子 →
自由なはずなのに、なぜか誰かに決めてほしくなる時。
エーリッヒ・フロム →
頭で分かっているのに、体や気持ちがついてこない時。
メルロ=ポンティ →
分かっているのに行動できない自分に、もどかしさを感じた時。
王陽明 →

⛓️「こうすべき」に縛られる・常識が苦しい時

世間の当たり前や役割に、息苦しさを感じる。

「こうすべき」に縛られて、自分の本音が言えなくなった時。
ニーチェ →
「これが常識」「これが普通」に、息苦しさを感じた時。
フーコー →
「普通はこうだ」と誰かに決めつけられて、納得できなかった時。
プロタゴラス →
物事を「良い/悪い」「勝ち/負け」で二つに割り切って苦しくなった時。
デリダ →
何かを信じ込んで、後から違ったと分かって落ち込んだ時。
ピュロン →
まわりの目が気になって、見栄のためにお金や時間を使ってしまう時。
ディオゲネス →
昔とは変わってしまった自分や関係に、寂しさを感じた時。
ヘラクレイトス →
周りに流されて、自分の意見を言えないまま終わった時。
アーレント →
先入観や決めつけで、物事をちゃんと見られていないと感じた時。
フッサール →
自分の考えを否定されるのが怖くて、意固地になった時。
ポパー →
何でもお金や損得で考える癖に、ふと虚しさを感じた時。
